# MongSrape
MongSrape
In this assignment, we created a web app that lets users view and leave comments on the latest news using mongo and cheerio to scrape news from another site.

## Technology used 
1. express
2. express-handlebars
3. mongoose
4. body-parser
5. cheerio
6. request
7. axios
